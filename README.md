# 견적서 품목 추출기

G마켓 견적서 엑셀 파일을 업로드하면, 브라우저 안에서 `내용`, `규격`, `수량`, `예상단가`를 추출하는 정적 웹사이트입니다.

추출은 컬럼 좌표가 아니라 키워드 매칭으로 처리합니다.

- `상품명`, `필수선택`, `추가구성` → `내용`
- `규격` → `개`
- `수량` → `수량`
- `예상단가` → `공급가액 ÷ 수량`
- `할인금액`은 계산에서 제외

업로드한 엑셀 파일은 서버나 외부 API로 전송하지 않습니다. 파일 내용은 사용자의 브라우저 안에서만 읽고 처리합니다.

## GitHub Pages 배포 방법

1. GitHub에서 새 저장소를 만듭니다.
2. 이 폴더의 파일을 저장소에 올립니다.
   - `index.html`
   - `styles.css`
   - `app.js`
   - `.nojekyll`
   - `README.md`
3. 저장소의 `Settings`로 이동합니다.
4. 왼쪽 메뉴에서 `Pages`를 선택합니다.
5. `Build and deployment`에서 다음처럼 설정합니다.
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
6. 저장하면 잠시 뒤 아래 형태의 주소가 생깁니다.

```text
https://사용자이름.github.io/저장소이름/
```

이 주소를 엣지 브라우저에서 열면 됩니다. PowerShell을 사용할 필요가 없습니다.

## 사용 방법

1. 견적서 엑셀 파일을 업로드합니다.
2. `추출하기`를 누릅니다.
3. 결과를 JSON으로 복사하거나 CSV로 저장합니다.

## 참고

현재 버전은 엑셀 파싱 라이브러리인 `SheetJS`를 CDN에서 불러옵니다. 이것은 프로그램 파일을 가져오는 용도이고, 업로드한 엑셀 내용 자체를 외부로 보내지는 않습니다.
